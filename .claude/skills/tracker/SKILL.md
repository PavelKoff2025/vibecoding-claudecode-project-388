# tracker

Собирает цены сразу по всему списку отслеживаемых товаров и отдаёт
единую таблицу прогона. Сам цену не парсит — для каждого URL вызывает
готовый скилл extract-price и переиспользует его результат.

## Отслеживаемые URL

Список зашит прямо в скилл, чтобы трекер запускался одной фразой
без ручного перечисления ссылок:

- https://www.chitai-gorod.ru/product/palacco-madamy-voobrazaemyj-muzej-iriny-antonovoj-3118334
- https://www.chitai-gorod.ru/product/proshchanie-s-illyuziyami-3035451
- https://www.chitai-gorod.ru/product/vozvrasenie-sinej-borody-3157566
- https://www.chitai-gorod.ru/product/dyagilev-s-dyagilevym-3022585
- https://www.chitai-gorod.ru/product/biblia-vina-3158884

## Главный сценарий

1. Взять список URL, зашитый в этот скилл (секция выше).
2. Для КАЖДОГО URL вызвать скилл extract-price и получить объект цены:

       {
         "regular_price": <число|null>,
         "sale_price": <число|null>,
         "has_credit": <true|false>
       }

   Tracker переиспользует extract-price и НЕ дублирует его логику
   парсинга. Вся работа с HTML, JSON-LD и нормализацией цены —
   ответственность extract-price.
3. К каждому объекту цены добавить исходный URL.
4. Собрать все объекты в единую таблицу прогона (см. ниже).
5. За один запуск обойти ВЕСЬ список — не по одному товару.

## Таблица прогона

Результат — одна таблица, одна строка на товар. Колонки строки:

| url | regular_price | sale_price | has_credit |
|-----|---------------|------------|------------|

Это будущая запись прогона: именно её позже будем сохранять в историю
и сравнивать между запусками (diff), а изменения слать в Telegram.

## Как выполнять

Для каждого URL из списка выше вызвать готовый скилл extract-price:

    python .claude/skills/extract-price/extract_price.py "<URL>"

Собрать stdout-объекты по всем URL в одну таблицу прогона
и вывести её целиком.

## Планы (следующие шаги)

- Хранение: сохранять таблицу прогона в файл истории.
- Diff: сравнивать текущий прогон с предыдущим, выделять изменения цен.
- Telegram: отправлять уведомление об изменениях.
