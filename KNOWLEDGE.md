# KNOWLEDGE.md — правила значимости изменений (tracker)

Товары сопоставляются по полю `url`. Для каждого сравниваются поля
`regular_price`, `sale_price`, `has_credit` между прошлым и текущим прогоном.

## Порог значимости цены
- Существенным считается изменение цены на ±1% и больше.
- Изменение меньше 1% — незначимое, отбрасывается.
- Проверяется и regular_price, и sale_price.

## Скидка (sale_price)
- null → число  = началась акция (значимое).
- число → null  = акция закончилась (значимое).
- изменился размер скидки = значимое, если >= 1%.

## Рассрочка (has_credit)
- false → true = появилась рассрочка (значимое).
- true → false = убрали рассрочку (значимое).

## Состав каталога
- новый url = новый товар (значимое).
- пропал url = товар снят (значимое).
